# Handover

## Purpose

Capture decisions, completed work, and next actions so a new session can resume quickly.

## Inputs

- session transcript context
- repo status and branch
- key files changed

## Process

1. Summarize what was done.
2. Record key decisions with rationale.
3. List unresolved questions.
4. Provide ordered next steps.

## Output Format

- Session Summary
- What Was Done
- Key Decisions
- Open Questions
- Next Steps

## Quality Checks

- [ ] Includes rationale, not only actions
- [ ] Notes blockers and risks
- [ ] Next step list is actionable
