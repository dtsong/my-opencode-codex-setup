# Handover Workflow

Goal: produce `memory/HANDOVER-<timestamp>.md` in the current workspace.

Script:
- `scripts/handover.sh`

Usage:
```bash
./scripts/handover.sh
./scripts/handover.sh --print
```

Sections:
1. Session Summary
2. What Was Done
3. Key Decisions and Why
4. Open Questions
5. Next Steps

Rules:
- Keep it factual and concise.
- Capture rationale and risks.
- Do not invent details.
